Setup P5JS

  1. Install Visual Studio Code

    Download and install Visual Studio Code from here.

  2. Set up a project folder

    Create a new folder on your computer where you want to store your p5.js projects. Open Visual Studio Code and go to File -> Open Folder, and select the folder you just created.

  3. Downloading extensions

    In Visual Studio Code, we use extensions to enhance our development experience. Follow these steps to download the necessary extensions:

    1. Click on the square icon on the left sidebar or press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac) to open the Extensions view.
    2. Search for "p5.vscode" in the Extensions view search box and install the extension by Sam Lavigne.
    3. Search for "Live Server" in the Extensions view search box and install the extension by Ritwick Dey.

    The "p5.vscode" extension provides p5.js project templates and helpful snippets, while the "Live Server" extension allows you to run your p5.js sketches locally in a browser.

Running P5JS

  1. Structure of a p5.js Project Folder

    A typical p5.js project folder includes the following files:

    • An HTML file (e.g., index.html) that contains the structure of your web page.
    • A JavaScript file (e.g., sketch.js) where you write your p5.js code.
    • (Optional) A CSS file (e.g., style.css) to style your web page.

    To simplify the creation of a p5.js project folder, you can use the p5.vscode extension:

    1. Press `Ctrl+Shift+p` (Windows/Linux) or `Cmd+Shift+p` (Mac) while in the directory which you want to create the project in.
    2. Search for 'Create p5.js project' and select it.
    3. Choose the folder where you want to create your p5.js project and select it.

    The "p5.vscode" extension will automatically create the project folder structure for you. This files included in the newly created project are: index.html, sketch.js, style.css, jsconfig.json and a libraries folder.

  2. Live Server

    With the "Live Server" extension, you can easily run your p5.js sketches locally:

    1. Open your p5.js project folder in Visual Studio Code.
    2. Find the HTML file (e.g., index.html) and right-click on it.
    3. Select "Open with Live Server" from the context menu.
    4. A new browser tab will open, displaying your p5.js sketch.

    Whenever you make changes to your p5.js code and save the file, the browser will automatically refresh and show the updated sketch.
