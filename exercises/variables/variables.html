Global/Local variables

Create a circle again using variables, however this time define the radius variable inside setup() - in another function.

What happens? the radius variable is defined within another function and is said to be out of scope so it cannot be used. Variables defined outside of any function are said to be globally defined, while variables defined within a function or {} braces (a block) are said to be defined locally.
