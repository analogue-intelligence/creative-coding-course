Basic Geometry

N. points

N. Lines

Draw a line which cross the canvas starting from the top-left to the bottom-right of the canvas

Rec - CENTER

Draw a rectangle using the CENTER rectangle mode, the rectangle is centred at (50, 50) with width 40, and height 40

Rec- CORNER

Draw a rectangle using the CORNER rectangle mode, the top-left corner is at (30, 30) with width 40 and height 50. Note that CORNER is the default rectangle mode

Ellipse - CORNER

Draw an ellipse using the CORNER ellipse mode, the ellipse should be centered at x-coordinate 50, y-coordinate 50 and has length 20 and height 40

Ellipse - RADIUS and CENTER

Draw two ellipses, both are centred at (50, 50) and both have width 30 and height 30, however one is in CENTER mode while the other is in RADIUS mode. Why are they different? In RADIUS mode the the width and height of the ellipse are actually half of the width and height, which is why we see the ellipse in RADIUS as twice the size of the on in CENTER mode. Note that CENTER mode is the default mode
