Summer school - Creative Coding

Exercises week 1

Stroke and Fill

Stroke and Fill

Create two circles, one circle with a black stroke and a white fill, and another circle with a white stroke and a black fill. The circles can be placed anywhere. Note that the default stroke colour is black (0) and the default fill colour is white (255)

noStroke() and noFill()

Draw two circles which are placed anywhere, one has a gray fill but no stroke, while the other circle has a gray stroke but no fill

Stroke Weight

Draw two circles which are placed anywhere, one has a stroke weight of 1, and the other has a stroke weight of 4

Colour

Colour-Fill

Create three circles which are placed anywhere, one is coloured red, another one is green and the last one is blue

Colour-Stroke

Create three circles which are placed anywhere, the fill of the circles is any colour you want. One has stroke coloured red, another one has stroke coloured green and the last one has stroke coloured blue

Alpha

Create three black circles which are placed anywhere, one has an alpha value of 20, another one has an alpha value of 140, and another one has an alpha value of 255

Mouse

Mouse

Draw a line centred at (50, 50) which follows the mouse, as an extra try to also create a circle which moves with the mouse

Pressing the mouse

Draw a circle at the current mouse location whenever the mouse has been pressed.

Interactive Line

Using the previous and current points of the mouse, draw a line which moves along with the mouse
