1.2 Basic Geometry

In this section we will learn how to draw different geometric figures, such as üü®, üü°, and more.

Let's first get started with the most basic one: point.

1.2.1 Point

Draw three points on the canvas using the following coordinates:

  • Point 1: x-coordinate 50 and y-coordinate 10.
  • Point 2: x-coordinate 10 and y-coordinate 90.
  • Point 3: x-coordinate 90, and y-coordinate 90.

1.2.2 Line

Draw a line which cross the canvas starting from the top-left to the bottom-right of the canvas

1.2.3 Rectangle - Center Mode

Draw a rectangle using the CENTER rectangle mode, the rectangle is centred at (50, 50) with width 30, and height 30

1.2.4 Recatangle - Corner Mode

Draw a rectangle using the CORNER rectangle mode, the top-left corner is at (30, 30) with width 40 and height 50.
Note: that CORNER is the default rectangle mode.

1.2.5 Ellipse - Corner Mode

Draw an ellipse using the CORNER ellipse mode, the ellipse should be centred at (50, 50) and has length 40 and height 40

1.2.6 Ellipse - Radius and Center Modes

Draw two ellipses, both are centred at (50, 50) and both have width 30 and height 30, however one is in CENTER mode while the other is in RADIUS mode.
Note: that CENTER mode is the default mode.

Why are the circles different? In RADIUS mode the width and height of the ellipse are actually half of the width and height, which is why we see the ellipse in RADIUS as twice the size of the one in CENTER mode.

1.2.7 Triangle

Draw a triangle with the first point at (20, 20), the second point at (25, 93), and the third point at (90, 10)

1.2.8 Arc

Draw a semi-circle using the arc() function, have the semi-circle centred at (50, 50), with width and height both at 80.
Note: the start and stop angles of an arc need to be specified in radians.

1.2.9 Quad

Draw a quad with the first point at (38, 31), the second point at (86, 20), the third point at (69, 63), and the last point (30, 76)

1.2.10 Curve

Draw a curve using the curve() function. Follow the instructions below:

  1. Create a canvas with a width of 100 and a height of 100.
  2. Set the background color to a light gray shade (e.g., background(220)).
  3. Draw a curve using the curve() function. The curve should have the following points:
  • Control Point 1: (mouseX, mouseY)
  • Point 1: (10, 50)
  • Point 2: (90, 50)
  • Control Point 2: (10, 10)

Additionally, draw the control points as circles using the ellipse() function. Connect the control points with lines using the line() function to visualize their influence on the curve.
