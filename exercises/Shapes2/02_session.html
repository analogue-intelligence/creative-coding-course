Summer school - Creative Coding

Exercises: Color

2.1 Stroke and Fill

2.1.1 Stroke and Fill

Create two circles, one circle with a black stroke and a white fill, and another circle with a white stroke and a black fill. The circles can be placed anywhere.
Note: the default stroke colour is black (0) and the default fill colour is white (255)

2.1.2 noStroke() and noFill()

Draw two circles which are placed anywhere, one has a grey fill but no stroke, while the other circle has a grey stroke but no fill

2.1.3 Stroke Weight

Draw two circles which are placed anywhere, one has a stroke weight of 1, and the other has a stroke weight of 4

2.2 Colour

2.2.1 Colour-Fill

Draw three circles which are placed anywhere, one is coloured red, another one is green and the last one is blue

2.2.2 Colour-Stroke

Draw three circles which are placed anywhere, the fill of the circles is any colour you want but all circles have the same colour. One has stroke coloured red, another one has stroke coloured green and the last one has stroke coloured blue

2.2.3 Alpha

Draw three circles of the same colour which are placed anywhere, one has an alpha value of 20, another one has an alpha value of 140, and another one has an alpha value of 255. Do not forget that the background should be added to the draw() function.
